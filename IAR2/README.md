### Program execution

To start the program first open webots (version 2022b) and load the provided world. Right-click on the robot and open the controller by clicking "Show Robot Window".

After ensuring that it is listening on port 2001, execute the IAR1_exercise.ipnyb cell labeled "Task 1.4" and click on run.

This should enable you to click on the simulated controller in your browser to make the robot spin.
